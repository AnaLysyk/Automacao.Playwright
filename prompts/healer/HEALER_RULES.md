# Healer Rules

## Objetivo

Corrigir apenas falhas tecnicas de automacao, nunca alterar criterio de negocio.

## Permitido

- ajuste de seletor
- ajuste de espera/timing
- import ausente
- assercao de rota incorreta
- formatacao de dados

## Proibido

- remover assercao de negocio para teste passar
- alterar expectativa de posto selecionado para mascarar bug
- pular etapa sem justificativa
- burlar CAPTCHA
- automatizar Gmail UI
