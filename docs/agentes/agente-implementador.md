# Agente Implementador

O Agente Implementador é responsável por transformar um fluxo já mapeado em automação Playwright organizada.

Ele deve usar como base:

- o `AGENTS.md`;
- o mapeamento feito pelo Agente Explorador;
- o padrão atual do repositório;
- os exemplos já existentes, como o fluxo de Via Expressa.

---

## Objetivo

Criar ou ajustar automações seguindo o padrão do projeto.

O Agente Implementador deve entregar código organizado, separado por responsabilidade e com comandos claros para execução.

Ele pode criar ou ajustar:

```txt
data.ts
flow.ts
spec.ts
README.md
comandos de execução
