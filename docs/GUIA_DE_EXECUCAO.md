# Guia de Execucao

## Pre-condicoes

- VPN conectada
- .env configurado
- URL do ambiente acessivel

## Passos

1. npm install
2. npx playwright install
3. npm run test:list
4. npm run test:cidadao:agendamento
5. npm run report:testing-company
