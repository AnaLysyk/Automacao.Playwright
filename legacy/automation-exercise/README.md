# Automation Exercise Legacy

Esta pasta guarda o projeto antigo do desafio Automation Exercise.

Ele foi preservado apenas como referencia didatica de Playwright, Page Object Model e massa dinamica. Nao faz parte da automacao principal de Booking / Cidadao Smart / SMART.

Conteudo movido:

- `pages/`
- `tests/e2e/automation-exercise.spec.js`
- `tests/api/automation-exercise-api.spec.js`
- `utils/gerador-dados.js`
- `playwright.config.js`

Para a automacao atual, use a estrutura TypeScript na raiz do repositorio.
