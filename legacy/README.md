# [LEGACY] Legacy

## Objetivo

Guardar projetos antigos ou materiais de referencia.

## Tipo de execucao

Nao faz parte da execucao principal.

## Comando principal

Nao ha comando principal para regressao.

## Pode entrar aqui

- Referencia didatica.
- Historico de aprendizado.
- Codigo antigo isolado.

## Nao pode entrar aqui

- Novos testes de Booking, Cidadao Smart ou SMART.
- Scripts principais do `package.json`.
- Credenciais ou evidencias locais.
