# [SUPPORT] Tests

## Objetivo

Concentrar a automacao executavel em Playwright.

## Tipo de execucao

Misto: regressao, assistido, POC e suporte. Use a legenda de cada pasta.

## Comando principal

```bash
npm run test:list
```

## Pode entrar aqui

- Specs Playwright.
- Page Objects.
- Agents.
- Helpers, massas, providers e relatorios de apoio.

## Nao pode entrar aqui

- Credenciais, `.env.local`, tokens ou massa sensivel.
- Evidencias locais de execucao.
- Projeto legado novo.

Legenda: `[REGRESSAO]` automatico, `[ASSISTIDO]` com QA, `[POC]` experimento, `[LEGACY]` antigo, `[SUPPORT]` apoio.
